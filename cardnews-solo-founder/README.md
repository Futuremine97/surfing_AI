# 한국에서 1인 창업 — 인스타 카드뉴스

`/Users/sunghoon/.gemini/antigravity/scratch/no-slip-saas`(No Slip SaaS) 프로젝트의
"작게 만들고 바로 쓰는" 스타일을 그대로 가져와, **"한국에서 1인 창업 정보"** 주제의
인스타그램 카드뉴스(1080×1080, 10장)를 만드는 작은 생성기입니다.

> 참고: 스크래치 폴더는 이 세션의 작업 디렉터리 밖이라 쓸 수 없어, 결과물을
> 현재 저장소 안의 이 폴더에 담았습니다. 통째로 옮기거나 지워도 됩니다.

## 빠르게 보기
- 브라우저로 **`index.html`** 열기 → 10장 미리보기 그리드.
- 각 카드 원본은 `cards/01.html` ~ `cards/10.html` (각 1080×1080).

## 내용 고치기
- 카피/문구: **`content.py`** 의 `CARDS` 수정.
- 인스타 핸들·태그라인: `content.py` 의 `BRAND` 수정 (`@1인창업노트` → 본인 계정).
- 디자인(색/폰트/크기): **`style.css`**.
- 고친 뒤 다시 만들기: `python3 generate.py`
  (이미 만들어진 `cards/*.html`, `index.html`, `style.css` 가 갱신됩니다.)

## PNG로 추출 (업로드용)
```bash
bash render.sh
```
Chrome/Chromium 헤드리스로 `png/01.png … png/10.png`(2160×2160, 2x)를 만듭니다.
Chrome이 없으면 `index.html`에서 카드를 열고 직접 캡처해도 됩니다.

## 카드 구성 (10장)
1. 표지 — 한국에서 1인 창업 시작하기
2. 왜 지금 1인 창업인가
3. STEP 1 · 사업자등록 (홈택스 5분)
4. STEP 2 · 간이과세 vs 일반과세
5. STEP 3 · 세금 캘린더 (1·5·7월)
6. 정부 지원 (K-Startup·소진공·1인 창조기업·청년창업사관학교)
7. 안전망 (노란우산공제·4대보험)
8. 1인 창업 아이템
9. 초보가 자주 하는 실수 체크리스트
10. CTA · 오늘 할 일 + 면책 고지
